# isometric

![Video demonstration](https://github.com/IanFeldman/procedural-2d-game/blob/main/screenshots/gameplay.gif)

Explore an infinite procedural world in this SDL2-powered game demo!

## Controls
| Key | Description |
| --- | --- |
| `w` | Move up |
| `s` | Move down |
| `a` | Move left |
| `d` | Move right |
